Redis Operator

A Kubernetes operator that automatically manages Redis 6.2 Cluster deployments, handling cluster bootstrap, scale-up, and scale-down operations.

Get Started View on GitHub

Features

Automatic Bootstrap

Automatically bootstraps Redis cluster when 3 or more pods are ready. No manual intervention required.

Scale Up

Seamlessly add new nodes to your cluster. The operator handles node joining and slot distribution.

Scale Down

Safely remove nodes with automatic slot migration and cluster forget operations.

Password Authentication

Secure your cluster with password authentication using Kubernetes Secrets.

Stable DNS Addressing

Uses hostname-based cluster announcement via StatefulSet pod names for stable addressing.

Continuous Reconciliation

Constantly monitors and reconciles cluster state to ensure desired configuration.

Quick Start

1

Add the Helm repository

helm repo add troke12 https://troke12.github.io/redis-operator
helm repo update
2

Install the operator

helm install redis-operator troke12/redis-operator \
  --namespace redis-operator \
  --create-namespace
3

Deploy a Redis cluster (optional)

helm install redis-operator troke12/redis-operator \
  --namespace redis-operator \
  --create-namespace \
  --set redisCluster.enabled=true \
  --set redisCluster.password='your-secure-password'
